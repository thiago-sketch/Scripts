===LEIA ANTES DE UTILIZAR===

ATENCAO: LEIA OS PASSOS A SEGUIR ANTES DE UTILIZAR O SCRIPT.
	
O QUE ESTE SCRIPT FAZ?	
	Este script ira habilitar a conta de administrador interna do Windows com a senha "@IM&&@B!v1" (sem aspas)
	e ira fornecer ao técnico a opção de remover o usuario de onde o script está sendo executado ou qualquer outro que ele deseje.
	
COMO USAR ESTE SCRIPT?
	1-	Se certifique de ter completada a instalação dos softwares basicos e feche qualquer tarefa que esteja rodando em segundo plano
	2-	Inicie o script como administrador
	3-	Selecione o idioma do sistema operacional (en-US para ingles e pt-BR para portugues) onde o script esta sendo executado
	4-	O script ira habilitar a conta administrador correspondente com a senha "@IM&&@B!v1" (sem aspas)
	5-	Apos ser habilitada com sucesso, o tecnico devera digitar o nome da conta administrador sem senha que ele deseja remover
	6-	Se o nome digitado estiver correto o usuario sera deletado, caso esteja errado e o tecnico queira tentar novamente,
		basta reiniciar o script e refazer o mesmo processo
	7-	Após o usuario ser deletado com sucesso, faça o logoff e entre no usuario desejado.
